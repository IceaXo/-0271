# 🛒 S.E.E.S. Mall - 简易电商系统

> 基于 Spring Boot + MyBatis-Plus + Thymeleaf 的 B/S 架构电商平台。
> 本项目为《网络应用开发》课程实验作业。

## 📖 项目简介

**S.E.E.S. Mall** 是一个轻量级的电子商务网站，实现了从用户注册、浏览商品、下单购买到邮件通知的完整购物流程，同时包含一个功能完善的后台管理系统，支持商品管理、订单处理及销售数据统计。
部署在线网站：     http://120.26.2.145:8080/    

## 🛠️ 技术栈

- **核心框架**: Spring Boot 3.1.5
- **持久层**: MyBatis-Plus 3.5.7
- **数据库**: H2 Database (嵌入式内存数据库，无需额外安装 MySQL)
- **模板引擎**: Thymeleaf
- **工具库**: Lombok, JavaMailSender (邮件服务)
- **构建工具**: Maven
- **JDK 版本**: Java 21

## ✨ 功能特性

### 👤 顾客端 (Client)
1.  **用户体系**：支持用户注册、登录、注销。
2.  **商品浏览**：首页展示所有上架商品。
3.  **购物流程**：
    - 商品库存实时扣减。
    - **异步邮件通知**：下单成功后，系统自动发送确认邮件，不阻塞页面跳转。
4.  **订单中心**：查看历史订单状态（待发货/已发货）。

### 🛡️ 管理员端 (Admin)
1.  **数据仪表盘**：登录后自动跳转至统计页，展示**最近 7 天销售额**趋势图。
2.  **商品管理**：发布新商品、编辑现有商品信息、下架商品。
3.  **订单管理**：
    - 查看全平台订单。
    - **一键发货**：点击发货后，自动触发邮件通知买家。
4.  **用户管理**：查看及管理注册用户。

## 🚀 快速开始 (Quick Start)

### 1. 环境准备
确保本地已安装：
- Java JDK 21+
- Maven 3.6+

项目默认使用 **H2 内存数据库**，启动即用，数据保存在内存中（重启后重置）。
